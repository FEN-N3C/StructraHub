**StructraHub** is a Roblox internal GUI and aimbot framework designed for fast, flexible, and clean gameplay enhancements. It utilizes Fluent UI to provide a sleek, customizable interface with fully functional aimbot controls.

## Aimbot
- Toggleable aimbot system
- Team check
- Dead check
- Wall check
- Friend check
- Prediction system
- Lock type
- Lock part
- Randomization: randomly switch between target parts at a configurable interval.
- Keybind support

## User Interface
- Built with Fluent UI for a modern and minimal design.
- Tabs for "Aimbot" and "Settings".
- Toggles, sliders, dropdowns, and keybinds are fully configurable.
- Configuration and UI state are saved automatically between sessions using the Fluent Addons:
- SaveManager
- InterfaceManager
